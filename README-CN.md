## 特征 ✨

- 两种生成封面的方式: 图文 和 纯色
- 图形 模式支持上传图片和使用 UnSplash 图片
- 纯色 模式支持自定义背景颜色。
- 支持可选字体
- 响应式设计，支持暗黑模式
- 多语言支持:（中文、英文）

## 如何使用

1. 在此处申请 Unsplash API 密钥 [这里](https://unsplash.com/developers).
2. 在根目录下创建一个 .env.local 文件
3. 参考 .env.example 文件，在 .env.local 文件中添加配置

## 安装

```bash
pnpm install

npm run dev
```
